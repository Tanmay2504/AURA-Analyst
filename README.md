# AI Data Analyst Agent

## One-click start

Use either of these options:

1. Double-click `start-dev.bat` in the workspace root.
2. In VS Code, run the task `Start Full Stack`.

That starts:

- Backend: `backend/venv/Scripts/python.exe -m uvicorn main:app --reload`
- Frontend: `frontend` via `npm run dev`

## URLs

- Backend docs: http://localhost:8000/docs
- Frontend app: http://localhost:3000 or the next available port shown by Next.js
